# 보고서 형식

## 1.담당자별
```
👤 담당자별 | 📅 {기간} | 👥 {팀}
| 담당자 | 전체 | 완료 | 진행중 | 대기 | 완료율 |
```
완료율 = 완료/전체×100

## 2.기간별
```
📉 기간별 | 📅 {기간} | 👥 {팀}
| 주차/월 | 발생 | 완료 | 누적미해결 | 증감률 |
```
누적미해결 = 이전누적+발생-완료

## 3.상태별
```
🔄 상태별 | 📅 {기간} | 👥 {팀}
| 상태 | 개수 | 비율 |
```

## 4.유형별
```
📋 유형별 | 📅 {기간} | 👥 {팀}
| 유형 | 건수 | 비율 | 대표이슈 |
## 상세: ### {유형} → - LPM-XXXX: 제목
```

## 5.영역별
```
🏷️ 영역별 | 📅 {기간} | 👥 {팀}
| 영역 | 건수 | 비율 | 분류 |
## 상세: ### {영역} ({N}건) [{태그|클러스터}] → - LPM-XXXX: 제목
```
