# 영역 분류 (하이브리드)

## 흐름
Step0: 팀명태그필터 → Step1: 태그추출 → Step2: 클러스터링 → Step3: 병합

## Step0: 팀명 필터
```
팀명목록: backend,payment,messaging,community,ios,android,개발팀,QA팀
태그가 팀명 → 무시, 제목으로 영역추출
예: [Backend] 투표 릴레이 → "릴레이"
```

## Step1: 태그추출
```
정규식: /^\[([^\]]+)\]/
정규화: 대소문자통일, trim, 팀명제외
```

## Step2: 클러스터링 (태그없는 이슈)
| 키워드 | 영역 |
|--------|------|
| 결제,정산,PG,환불 | 결제 |
| API,배치,hotfix,서버 | API/서버 |
| 채팅,메시지,알림 | 채팅 |
| 통계,집계,리포트 | 통계 |
| UI,디자인,화면 | UI/UX |
| QA,테스트,검증 | QA |

새패턴→자동생성 | 분류불가→기타

## Step3: 병합
태그+클러스터 통합 → 3건미만 병합검토 → 건수 내림차순
